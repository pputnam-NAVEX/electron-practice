The is a very basic build/framework for Electron utilizing node and node package manager (NPM), it displays "Hello World" and the version of node, Chrome, and Electron that the system that it is run on is using.

**************************************
Tutorial Concepts & files
**************************************
- Setup new application process
- package.json setup
- index.html
- main.js
- preload.js
- renderer.js (empty, but linked)

**************************************
Installation
**************************************

After cloning into a directory, navigate to that directory in Bash command prompt and type:

$ npm install

**************************************
Information about application
**************************************

Other than main NPM components from the "npm init" command, the only installed package install command was:

$ npm install --save-dev electron
